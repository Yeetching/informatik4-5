Instructions:

1- Sélectionnez votre mode désiré entre les modes facile et difficile. Votre but est de survivre les attaques par les memes le plus de fois possible.

2- Peu importe le mode choisi, vous aurez toujours 200 points de vie.

3- Une fois dans le jeu, choisissez un objet entre roche, papier et ciseaux, et cliquez sur "Fight!"

4- Si vous gagnez une partie, vous gagnez le nombre de vie équivalent à un chiffre aléatoire entre 0 et 5 points de vie. Si vous égalisez, rien ne se passe et si vous perdez, vous perdrez le nombre de vie équivalent à un chiffre aléatoire selon le mode choisi: facile = entre 0 et 50 points de vie, difficile = entre 0 et 100 points de vie. Notez que votre nombre total de vies ne peut dépasser 200.

5- Lorsque votre nombre de vies tombe à 0, vous avez perdu la partie. On affichera votre nombre d'attaques survécu par les memes.

6- Game on!
